## Build

Build Exile is very simple,please click Exile.sln if your installed Visual Studio Community 2013 or 2015 ,


```bat
git clone https://github.com/fstudio/Exile.git
cd Exile
nuget restore
msbuild Exile.sln
```

```bat
git clone https://git.oschina.net/ipvb/Exile.git 
cd Exile
nuget restore
msbuild Exile.sln 
```
